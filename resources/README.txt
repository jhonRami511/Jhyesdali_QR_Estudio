DESCRIPCIÓN:
Aplicación de escritorio para generar códigos QR personalizados,
diseñada especialmente para entornos de control de inventarios,
etiquetado de productos y gestión de archivos.

CARACTERÍSTICAS:
• Generación de códigos QR desde texto o enlaces.
• Configuración de tamaño, color, bordes y logotipo central.
• Opción de vista previa y guardado en formato PNG, JPG, PDF o SVG.
• Compatible con lectura estándar en dispositivos Android y PC.
• Para algunos modelos de iPhone, la cámara nativa puede no detectar
  el QR; en estos casos se recomienda usar una app de terceros.

REQUISITOS:
• Sistema operativo Windows 10 o superior.
• Lector de códigos QR (nativo o externo en móviles).

INSTRUCCIONES DE USO:
1. Abra "Jhyesdali QR Estudio".
2. Ingrese el texto o enlace a codificar.
3. Configure los parámetros de color, tamaño y logo si lo desea.
4. Presione "Generar QR" para visualizar el resultado.
5. Use “Guardar como” para exportar el código.
